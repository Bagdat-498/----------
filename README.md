📚 Учебная система (School System)

Это веб-приложение для авторизации и работы пользователей с разными ролями.

🚀 Возможности
	•	🔐 Регистрация и вход
	•	👨‍🏫 Роль учителя
	•	👨‍🎓 Роль студента
	•	🔄 Перенаправление на разные страницы после входа
	•	💾 Сохранение токена и имени пользователя

🛠️ Технологии
	•	HTML
	•	CSS
	•	JavaScript
	•	Node.js (сервер)

📂 Страницы
	•	login.html — вход
	•	signin.html — регистрация
	•	teacher.html — страница учителя
	•	student.html — страница студента
